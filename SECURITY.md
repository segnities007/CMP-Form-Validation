# Security Policy

## Reporting a Vulnerability

Please do not open public issues for security vulnerabilities.

Report vulnerabilities via GitHub Security Advisories:

- https://github.com/segnities007/CMP-Form-Validation/security/advisories/new

## Response Process

- We will acknowledge the report and begin triage.
- We will assess severity and impact.
- We will prepare and release a fix when confirmed.
- We may request additional details to reproduce the issue.

## Scope

Security reports are welcome for:

- `validation-core`
- `validation-compose`
- CI/CD configuration and dependency supply chain
