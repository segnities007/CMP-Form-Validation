# validation-compose

Compose integration module for `validation-core`.

## Scope

- `rememberValidatedField`
- `rememberValidatedStringForm`
- `Modifier.validation`
- `ValidationSupportingText`
- `ValidatedOutlinedTextField`

## Notes

- Primary API is explicit state binding with native `TextField`/`OutlinedTextField`.
- Wrapper/modifier APIs are supplementary.
